# CodeLeap Network

Very simple application that will perform basic CRUD functions.

- [Live Link](https://codeleap-network-43s5.vercel.app/)
## Stack

**Front-end:**
- Vite
- ReactJS
- TypeScript
- CSS
- Redux
- React UUID
- Phosphor React
- React Router DOM
- Redux Persist
- Axios
- React Transition Group



## Features

- Enter your User Name
- Sign Out
- Create a Post
- Edit only your posts
- Delete only your posts
- Like posts

## Running Locally

Clone the project

```bash
  git clone https://github.com/helioLJ/codeleap-network
```

Enter the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start your local server

```bash
  npm run dev
```
